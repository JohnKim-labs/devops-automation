# 🧭 Studio DROX Maintenance Reports Repository

이 저장소는 외부/내부 프로젝트의 유지보수 보고서를 관리하는 GitHub 기반 백업 허브입니다.

## 구조
- `external/` : 외부 고객 프로젝트 보고서 (youhak.com 등)
- `internal_winnie/` : Winnie 플랫폼 서비스 보고서
- `internal_drox/` : Studio DROX 자체 앱 보고서
- `notion_sync/` : Notion DB 스키마 및 동기화 로그

자동화된 Claude Code Skills 시스템이 이 파일들을 스캔하여 요약, PDF 리포트, Notion 업데이트를 수행합니다.
